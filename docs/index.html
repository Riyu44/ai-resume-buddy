How It Works

  1. Upload your resume as a PDF — text is extracted with PyPDF2.
  2. Paste any job description into the text area.
  3. A structured prompt sends both to Gemini 1.5 Flash and requests a JSON response.
  4. The app parses the JSON and renders your fit score, keyword gaps, and rewritten bullets.
